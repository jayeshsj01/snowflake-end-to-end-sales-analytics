# 📊 Power BI Dashboard

This folder contains the Power BI dashboard developed for the **Snowflake End-to-End Sales Analytics** project.

The dashboard connects with the Snowflake data warehouse and provides interactive insights into sales performance through filters, KPIs, and visualizations.

## Features

- Sales Overview Dashboard
- Interactive Date, Category, and Region Filters
- KPI Cards
- Store Type Analysis
- Region-wise Sales Analysis
- Category-wise Sales Analysis
- Loyalty Program Insights

## Files

- `Dashboard_Overview.png` – Complete dashboard
- `Dashboard_Category_Filter.png` – Category filter demo
- `Dashboard_Region_Filter.png` – Region filter demo
- `Dashboard_Date_Filter.png` – Date filter demo

> Built using **Power BI**, connected to **Snowflake**, and powered by SQL-based data modeling.
