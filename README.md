# ❄️ Snowflake End-to-End Sales Analytics

> **An end-to-end Data Warehouse project built using Snowflake, Python, SQL, and Power BI. This repository demonstrates data generation, ingestion, transformation, business-oriented SQL analysis, and interactive dashboard development.**

---

## 🚀 Project Overview

This project simulates a real-world retail sales analytics solution by building an end-to-end data warehouse on Snowflake. The workflow covers data generation, loading CSV files into Snowflake, designing a Star Schema, performing business-driven SQL analysis, and creating interactive Power BI dashboards.

The repository also includes real-world SQL business scenarios, detailed explanations, and project documentation to showcase practical data engineering and analytical skills.
